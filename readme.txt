
1)Implementation
__________________

This solution uses Python 3.

To run this solution, install python3 first. More information on how to install python3 for your OS are found in: https://www.python.org/downloads/

Put all the python files in the same directory

2)Command execution
___________________

To run the application, use (for instance):

python3 ~/path_to_app/unbabel_cli.py --input_file events.json --window_size 10

for guidance on how to use the application, run:

python3 ~/path_to_app/unbabel_cli.py --help

The output files will be stored in a new directory inside the directory where the python scripts are.
